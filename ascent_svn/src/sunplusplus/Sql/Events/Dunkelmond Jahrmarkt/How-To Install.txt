How-to install:

1. Suche dir einen Event Platz aus! (Mulgore, Wald von Elwyn, Wälder von Terrokar)

2. Öffne den Ordner des gewählten Standortes

3. Batche zuerst X_SettingUp.sql und dann X_ActiveEvent.sql

4. Nun ist das Event aktiviert

5. Um das Event wieder zu löschen in Ordner "Remove" gehen, zuerst X_SettingUp_Remove.sql und dann X_ActiveEvent_Remove.sql batchen
